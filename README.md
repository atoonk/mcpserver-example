# mcpserver-example

A minimal FastMCP server demo with HTTP streaming (/shttp/) and SSE (/sse/).

## Requirements
- Python 3.10+
- fastmcp, uvicorn (`pip install fastmcp uvicorn`)

## Setup
```bash
python -m venv venv && source venv/bin/activate
pip install fastmcp uvicorn
```

## Run
```bash
python main.py
```
